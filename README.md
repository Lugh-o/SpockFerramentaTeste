# Como executar o projeto

## 1. Pré-requisitos

- Instalar Java (JDK 21 ou superior)

## 2. Clonar
```
git clone git@github.com:Lugh-o/SpockFerramentaTeste.git
cd [pasta-onde-foi-feito-o-clone]
```


## 3. Build

Linux:
```
./gradlew build
```

Windows:
```
gradlew build
```

## 4. Testes

Linux:
```
./gradlew test
```

Windows:
```
gradlew test
```
